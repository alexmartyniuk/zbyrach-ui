# Zbyrach

Zbyrach is a web application that helps users find articles and news by relevant keywords and sends found articles by e-mail.

A keyword is a word or concept that may interest a user. For example, *history*, *software developmen* or *HTML*. Currently, Zbyrach only supports tags in English language.

In order to use Zbyrach user need to have a Google account, an authorization is done through it. Zbyrach only gets the user's email address, name and photo from Google.

You can find working application at [http://zbyrach.martyniuk.info/](http://zbyrach.martyniuk.info/)
