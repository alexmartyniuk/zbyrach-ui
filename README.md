# Zbyrach

Zbyrach is a web application that helps users find articles and news by relevant keywords and sends found articles by e-mail.

![](https://github.com/alexmartyniuk/zbyrach-ui/blob/master/docs/img/web-ui-articles.png?raw=true)

A keyword is a word or concept that may interest a user. For example, *history*, *software developmen* or *HTML*. Currently, Zbyrach supports only keywords in English language.

In order to use Zbyrach user need to have a Google account. During authentication Zbyrach gets the user's email address, name and photo from Google.

You can find working application at [http://zbyrach.martyniuk.info/](http://zbyrach.martyniuk.info/)
